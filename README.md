# Cosmological Model-Independent Constraints on the Baryon Fraction in the IGM

This repository contains code to reproduce **Figure 1** (constant $f_{\rm IGM}$ case) of  
Lemos et al. (2023), using fast radio bursts (FRBs) and Pantheon Type Ia supernovae data.

## Files

- `model_independent_frbs01.ipynb` : Notebook performing the analysis and generating Figure 1.
- `lcparam_full_long.txt` : Pantheon SNe data file used in the notebook.
